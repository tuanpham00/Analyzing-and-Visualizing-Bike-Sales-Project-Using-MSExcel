# Analyzing and Visualizing Bike Sales Project Using MS Excel
# Overview
This project involves cleaning, analyzing, and visualizing data on bike sales.
# Objectives
- Present a comparison between average income and car purchase decisions by gender.
- Identify the age group of customers who purchase the most.
- Explore and categorize purchases based on specific criteria and keywords.
